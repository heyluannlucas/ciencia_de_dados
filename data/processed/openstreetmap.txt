OpenStreetMap is an open map being used by millions of devices and users every day. It can both be edited and used by anyone free of charge. This list contains projects using OpenStreetMap data for creative purposes, as well as projects dedicated to improving OpenStreetMap. We also have a list of unmaintained projects. If you are a developer, consider adopting one! (Source Code) OSM Inspector - The OSM Inspector (alias OSMI) is a web based debugging tool for advanced OpenStreetMap users offered by Geofabrik. Nominatim QA - The Nominatim Data Analyser is a QA tool used to scan the nominatim database and extract suspect data from it. These data are then presented to mappers through a visual interface so that they can correct them directly.
